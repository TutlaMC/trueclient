# TrueClient

TrueClient is a Minecraft developed by Tutla Studios

## Develop

1. Download off github
```bash
git clone https://github.com/TutlaMC/trueclient.git
cd trueclient
```

2. Install Dependencies
```bash
npm install
```

3. Test/Build

Test:
```bash
npm run dev
```
 
Build (Linx/MacOS):
```bash
npm run build
```

Build (Windows):
```bash
npx electron-builder --win
```
